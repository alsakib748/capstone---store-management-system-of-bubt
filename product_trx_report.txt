I've implemented the complete Product Lifetime History Report module. Here's a summary
  of what was created:
     
  Files Created
  1. Service Class
  app/Services/Reports/ProductLifetimeReportService.php

  - calculateOpeningStock() - Calculates stock before from_date
  - calculatePurchaseSummary() - Purchase qty, value, avg price
  - calculateIssueSummary() - Issue qty
  - calculateDamageSummary() - Damage qty & value
  - calculateReturnSummary() - Return qty
  - calculateClosingSummary() - Final stock calculation
  - generateFinalReport() - Combined report generation

  2. Controller

  app/Http/Controllers/Backend/ProductLifetimeReportController.php

  - index() - Show report page with filters
  - generate() - Generate report via POST
  - ajaxGenerate() - AJAX report generation
  - exportPdf() - Placeholder for PDF export

  3. Routes (in routes/web.php)

  /product/lifetime/report          → Index page
  /product/lifetime/report/generate → POST generate
  /product/lifetime/report/ajax     → AJAX endpoint

  4. Blade View

  resources/views/admin/backend/report/product_lifetime_report.blade.php

  Professional ERP-style UI with:
  - Filter form (from_date, to_date, product, semester)
  - Opening Summary section
  - Current Period Summary (Purchase, Issue, Damage, Return)
  - Final Closing Summary with stock formula
  - Detailed transaction history tables
  - Print functionality

  5. Sidebar Menu

  Added "Product Lifetime History" menu item in the sidebar.

  Business Logic Implemented

  Opening Stock Formula:
  Opening Stock = Previous Purchase - Previous Issue - Previous Damage + Previous Return

  Closing Stock Formula:
  Closing Stock = Opening Stock + Purchase Qty - Issue Qty - Damage Qty + Return Qty

  Stock Value:
  Stock Value = Closing Stock × Average Purchase Price

  The report is accessible from the Reports menu in the sidebar.
